# Nedlia DaVinci Resolve Plugin

Plugin for DaVinci Resolve to add and manage product placements.

## Features

| Feature       | Description                             |
| ------------- | --------------------------------------- |
| Time Marking  | Add product placements with time ranges |
| Control Panel | Manage placement properties             |
| Server Sync   | Save placements to Nedlia server        |
| File Export   | Generate placement data files           |

## Installation

1. Download the plugin from Nedlia portal
2. Copy to DaVinci Resolve scripts folder
3. Restart DaVinci Resolve

## Usage

1. Open a project in DaVinci Resolve
2. Go to Workspace → Scripts → Nedlia
3. Add product placements with time ranges
4. Sync to server when complete

## Requirements

- DaVinci Resolve 18+
- macOS 13.0+ / Windows 10+
- Nedlia account
