# Nedlia LumaFusion Plugin

Plugin for LumaFusion to add and manage product placements.

## Features

| Feature       | Description                             |
| ------------- | --------------------------------------- |
| Time Marking  | Add product placements with time ranges |
| Control Panel | Manage placement properties             |
| Server Sync   | Save placements to Nedlia server        |
| File Export   | Generate placement data files           |

## Installation

1. Download from App Store (when available)
2. Enable in LumaFusion settings
