# Nedlia Final Cut Pro Plugin

Plugin for Final Cut Pro to add and manage product placements.

## Features

| Feature       | Description                             |
| ------------- | --------------------------------------- |
| Time Marking  | Add product placements with time ranges |
| Control Panel | Manage placement properties             |
| Server Sync   | Save placements to Nedlia server        |
| File Export   | Generate placement data files           |

## Installation

1. Download the plugin from Nedlia portal
2. Copy to `/Library/Application Support/Final Cut Pro/Plugins/`
3. Restart Final Cut Pro

## Usage

1. Open a project in Final Cut Pro
2. Select a clip in the timeline
3. Open Nedlia panel (Window → Extensions → Nedlia)
4. Add product placements with time ranges
5. Sync to server when complete

## Requirements

- Final Cut Pro 10.6+
- macOS 13.0+
- Nedlia account
