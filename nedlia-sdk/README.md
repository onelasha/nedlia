# Nedlia SDKs

Client SDKs for integrating with the Nedlia platform.

## Available SDKs

| SDK           | Language   | Purpose                            |
| ------------- | ---------- | ---------------------------------- |
| `javascript/` | TypeScript | Video player integration (web)     |
| `python/`     | Python     | Server-side API integration        |
| `swift/`      | Swift      | iOS/macOS video player integration |

## Which SDK to Use?

| Use Case                | SDK        |
| ----------------------- | ---------- |
| Web video player        | JavaScript |
| iOS/macOS app           | Swift      |
| Server-side integration | Python     |
| Backend automation      | Python     |
